本周摘要

电力基建获制度性催化，但利润验证与效率革命正从两端挤压AI估值叙事

1. 本周研判

◆主判断

AI产业处于“基建狂潮遭遇回报拷问”的相持阶段。上游电力与先进封装获得政策和资本背书，但硬件利润率压缩、企业AI成本管控、BIS融资风险警告共同把市场从“信仰驱动”推向“证据驱动”。

◆要点
  • 短期主线：优先关注有制度红利、长期合约或自备发电能力的数据中心电力/散热/物理基建。
  • 中期约束：硬件毛利收缩、存储远期供给释放、AI收入与CapEx剪刀差，是估值修正的关键压力。
  • 关键窗口：未来1-3个月云巨头财报季。若AI收入增速继续落后于资本开支增幅，BIS所提示的“融资-回报断裂”可能从尾部风险变成基准情景。
2上期回顾展开
◆本节判断

本期基调从“未评估/多空分化”转向“偏乐观”，但乐观主要集中在电力基建、AI Agent工具链和先进封装/HBM，企业AI应用被下调至观察等待。

◆要点
  • 核心上调：白宫NEPA改革和2040亿美元电力并购额，将数据中心电力/散热/物理基建从“谨慎看好”推升至“积极看好”。
  • 结构变化：ASIC/推理加速由“观察等待”进入“谨慎看好”，驱动力来自中国国产芯片份额预计突破50%和推理主导权转移。
  • 主要降调：企业AI应用因微软低价捆绑与股东诉讼并存，从“谨慎看好”下调至“观察等待”。
  • 市场验证：MSFT、CRM、SNOW短期反弹验证软件侧弹性；CEG周跌11%与营收环比增103.7%形成背离；MU、AVGO下跌继续验证高估值硬件压力。

上期方向×本周市场表现回溯

软件侧反弹与硬件侧杀估值并存，电力基建政策利好尚未完全传导至股价。这意味着产业逻辑与市场情绪之间存在可交易的时间差，但不能把短期股价背离直接等同于基本面证伪。

3. 产业全景

AI产业链本周呈现“基建向上、应用求证、硬件承压”。NEPA改革和美国公用事业并购额上升，确认电力成为算力交付的新瓶颈；戴尔AI服务器毛利率降至17.8%，显示硬件“增收不增利”；KPMG调查和微软股东诉讼说明应用端进入ROI验证阶段。此时最确定的不是“AI车流量会多大”，而是“路仍会继续修”，但修路成本与回报证明开始变得同样重要。

4. 3个重要趋势信号

◆要点
  • 电力升级为国家级监管优先事项：NEPA改革引入195项分类排除，FERC要求六大电网运营商60天内改革并网流程。拥有自备发电能力或长期固定电价合同的数据中心资产更具竞争优势。
  • AI硬件增收不增利被财报验证：戴尔AI服务器营收增长，但整体毛利率降至17.8%，利润分配更偏向上游芯片和下游降本工具链。
  • 效率革命进入预验证阶段：LoopWM显示FLOPs下降25倍、长时程推理计算节省约100倍，叠加OECD训练效率提升约35%的数据，提示算力叙事可能被效率提升重估。

5. 产业链传导与前瞻推荐

链链条一：NEPA改革 + FERC并网命令 → 数据中心交付周期压缩 → 天然气发电与核电需求跃升 → 电力基建估值重估。

NEPA改革把大型电力项目审批从“不可预测的数年”压缩进195项分类排除和紧急程序，直接降低数据中心交付不确定性；Joulent 2.67GW天然气发电为微软数据中心供电，说明自备发电从备选变主流。投资含义是自备发电、长期PPA、核电和天然气设备/EPC的现金流质量提高；验证节点是云巨头数据中心交付指引与发电设备商订单积压。

链链条二：AI效率提升 → 同等算力产出倍增 → 推理芯片需求结构变化 → 英伟达生态护城河面临边缘侵蚀。

LoopWM给出FLOPs下降25倍、长时程推理计算节省约100倍的预验证信号，OECD称过去12个月训练效率提升约35%。若3-6个月内被独立复现，ASIC、边缘推理和开源模型组合的性价比会提高，英伟达估值中对CUDA锁定和算力线性增长的假设需要重估。

6. 冷门机会推断

◆冷门机会

AI推理的“去中心化”可能推升边缘计算基础设施需求，但该方向仍需真实部署数据验证。

◆要点
  • 逻辑：效率革命 + 小模型突破 + 推理主导权转移，可能使推理从“云端集中”走向“边缘分布”。
  • 触发条件：Salesforce、SAP等主要企业软件公司公开支持本地推理部署，或Dell等边缘服务器厂商披露AI相关收入高增。
  • 证伪信号：LoopWM复现失败，或小模型在真实生产环境中显著弱于大模型。
  • 跟踪抓手：Dell边缘产品线、IDC/Gartner边缘AI基础设施季度报告。

1. 宏观环境与政策风向：地缘缓和与政策松绑并行，能源板块意外成为AI宏观叙事支点

展开
◆本节判断

地缘风险短期缓和，但能源成本的中期支撑并未解除；NEPA改革才是对美国AI基础设施更有长期意义的变量。

◆要点
  • 关键证据：美伊谅解备忘录推动布伦特原油回落至$72.10/桶，但霍尔木兹海峡保险成本恢复仍需数月，60天后和平安排也未确定。
  • 政策催化：2026年6月30日白宫NEPA改革引入195项分类排除与紧急程序，加速AI数据中心、先进核反应堆和关键矿产审批。
  • 产业传导：审批周期一旦压缩12-18个月，数据中心交付风险下降，核电、天然气、电网资产成为最直接受益方。
  • 早期信号：Kioxia第十代NAND量产提示存储供应链从韩系双寡头向日本扩展，有助于分散亚太地缘集中度。

产业链传导与前瞻推演

链链条一：NEPA改革→许可加速→数据中心与核电交付周期压缩→算力供给改善。

NEPA改革将审批从逐案博弈转为195项明确豁免的制度框架，若大型项目审批压缩12-18个月，云厂商CapEx执行效率会提升，Constellation Energy(CEG)等核电运营商也可能提前释放新机组经济性。

链链条二：霍尔木兹海峡风险→能源成本→数据中心运营利润率→电力基础设施估值重构。

布伦特回落至$72.10/桶不等于能源成本压力解除，航运保险成本和PJM高峰调度仍会影响数据中心利润率；拥有自备发电或长期固定电价合同的运营商更具韧性。

链链条交汇：NEPA加速供给，能源成本抬升考验运营利润。

短期市场看“数据中心能否建起来”，中期看能源成本是否侵蚀利润率；净方向仍利好北美电力基建、核电和天然气板块，但依赖现货电力的二三线运营商需要更谨慎。

2. 产能瓶颈与供应链重构：电力并网成为算力交付首要约束，存储供给纪律确认

展开
◆本节判断

最紧约束已经从硅转向电力。FERC并网改革、自备发电模式和欧洲电网审批提案，共同确认“缺电”是算力交付的首要瓶颈。

◆要点
  • 电力侧：FERC要求PJM、MISO等六大运营商60天内改革输电服务、成本转移、共址发电等流程。
  • 自备发电：National Grid投资17.5亿美元支持Joulent 2.67GW天然气发电项目，为微软数据中心供电，说明“跨电表”模式正在主流化。
  • 存储侧：美光计划约2000亿美元制造和研发投入，供应紧张预计持续至2027年以后；但远期产能集中释放可能压制2028年后利润率。
  • 净效应：短期优先电力基建和先进封装；中期关注存储供给过剩信号和Meta入云对算力定价的重塑。

产业链传导

链「电网互联—自备发电—数据中心交付」链。

EMEA地区78GW已宣布数据中心项目能否落地，关键在电力可用性；英国平均电网连接等待时间超8年。VoltaGrid获黑石和哈利伯顿10亿美元融资，订单积压约7.5GW，客户包括Oracle 2.3GW，说明受益环节从半导体扩散至燃气轮机、模块化发电和EPC工程服务。

链「存储扩产—设备争夺—代工格局分化」链。

三星与SK海力士合计约800万亿韩元新建四座晶圆厂，ASE据报对CoWoS和FoCoS报价上调超20%，OSAT产能利用率接近满载。扩产增强设备商订单能见度，但也挤压二线晶圆厂设备可得性，并埋下2027年后供给集中释放风险。

链「Meta入云—算力供给格局—资本开支合规」链。

Meta拟把约20GW存量和14GW在建数据中心容量转化为云服务收入，会压制中立AI云服务商估值，但也为超大规模企业提供把CapEx转成可验证收入的路径。若模式成立，AI基建估值被质疑的风险会阶段性下降。

链链条交汇与净效应。

FERC命令、自备发电和欧盟审批提案共同确认电力约束主导短期逻辑；存储涨价与先进封装满载支撑中期供给侧；Meta入云则可能重塑云算力定价。短期优先电力基建和先进封装，中期跟踪存储供给过剩和云服务格局变化。

3. AI商业化落地与价值验证：成本约束显性化，硬件利润率与国产替代成为新焦点

展开
◆本节判断

AI商业化从“收入故事”进入“利润验证”。企业端开始管控AI支出，硬件端出现增收不增利，利润池向上游芯片和降本工具链重新分配。

◆硬观点

本周最核心的信号是，AI硬件业务的单位经济模型恶化与企业AI支出的预算天花板同时显现，构成了一个“增收不增利”的阶段性困局。

◆要点
  • 企业成本：KPMG调查显示近半数企业因成本缩减AI代理使用，开源模型可降低约70%部署成本。
  • 硬件利润：戴尔AI服务器已占总营收37%，但整体毛利率较去年同期下降约26个百分点。
  • 国产替代：科智咨询预计2026年国产芯片在中国智算市场份额首次超过50%，推理算力成为主战场。
  • 产业传导：AI服务器OEM利润率压缩；Google Claude Apps Gateway类成本治理工具价值上升；ASIC与国产芯片获得更有利竞争窗口。
  • 边缘观察：Arena商业服务8个月后ARR达1亿美元，第三方中立评测工具出现独立商业化信号；存储合约价续涨但增速放缓，需警惕非AI需求疲软。

4. 资本流向与估值逻辑：融资落地与系统性警告并行，资本叙事出现裂痕

展开
◆本节判断

资本没有撤离AI基础设施，但正在从无差别乐观转向“按现金流、合同和融资结构筛选项目”。

◆硬观点

资本并非撤离，而是在进行一场“内部筛选”——抛弃难以验证回报的炒作部分，向具有明确合同保障和物理资产支撑的领域集中。融资工具的创新（私人信贷）支撑了产业扩张，但BIS点出的结构脆弱性，又是远期最重要的宏观尾部风险。

◆要点
  • 正面信号：Fleet Data Centers完成46亿美元优先担保票据融资，用于内华达230MW数据中心园区；项目100%预租，租户为AA级信用主体，租期超过16年。
  • 扩张预期：摩根大通上调2030年全球AI资本支出预测至约5.5万亿美元，其中约4.1万亿需要债务融资；微软2026财年CapEx预计同比增长约61%至1900亿美元。
  • 风险警告：BIS指出私人信贷、循环融资和融资集中度风险。云商持有AI实验室股权、AI实验室再购买云算力的闭环，可能隐藏多层风险。
  • 验证节点：未来2-4周云巨头财报。若AI收入增速不能匹配资本开支增幅，BIS的“回报证伪”逻辑将更强地进入市场定价。
