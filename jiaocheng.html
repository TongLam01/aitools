如何获取并使用 API Key？

只需简单几步，即可为您的 AI 工具注入灵魂。

1

访问 DeepSeek 开放平台

使用浏览器访问 DeepSeek 官方开发者后台。

platform.deepseek.com 立即前往
2

注册并充值

新用户注册后有时会有免费测试额度，但还是建议你至少充值1元。

3

创建 API Key

在左侧菜单点击 "API Keys"，然后点击 "创建 API key"。

⚠️ 关键提示：

API Key 生成后仅显示一次。请立即复制并妥善保存到记事本中，切勿泄露给他人。

4

填入工具并开启对话

回到需要用到AI的工具页面，在设置区域找到 API Key 输入框，粘贴保存即可。

sk-xxxxxxxxxxxxxxx...
已保存

完成配置后，您的操作将直接调用 DeepSeek 官方接口。

常见问题 FAQ

接口调用报错提示“余额不足”？

这通常意味着您的免费额度已过期或已用完。您可以前往 DeepSeek 开放后台进行小额充值（通常 5-10 元即可使用很久）。

API Key 安全吗？

本工具为单 HTML 离线版本/前端调用，你的 Key 仅保存在你当前浏览器的本地缓存中，不会上传到任何第三方服务器。
